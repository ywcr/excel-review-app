测试2：复制长代码

测试复制较长的自动化代码：

// 自动化测试代码
const data = [
    { name: '张三', age: 25 },
    { name: '李四', age: 30 },
    { name: '王五', age: 28 }
];

function processData() {
    data.forEach((item, index) => {
        console.log(`处理第${index + 1}个用户: ${item.name}, 年龄: ${item.age}`);
    });
    console.log('数据处理完成！');
}

processData();

测试说明

  1. 点击"复制测试代码"按钮
  2. 按F12打开开发者工具
  3. 切换到Console标签
  4. 粘贴代码（Ctrl+V）并按回车
  5. 查看代码是否正确执行
